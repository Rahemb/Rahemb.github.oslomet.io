
## Description

This is my first ever website, created during my first semester at OsloMet. It showcases basic web development skills using HTML, CSS, and JavaScript.

## Features

- Home page with navigation
- Canteen information page
- Library resources page
- Accessibility page
- Topical content section

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to View

Simply open `HTML/index.html` in your web browser to view the website.

## Project Structure

- `HTML/`: Contains all HTML pages and styles
- `css/`: Additional CSS files
- `js/`: JavaScript files
- `Bilder/`: Images used in the website
